# Trendsight — 网络舆情事件智能分析系统

针对突发公共事件和社会热点话题，实现多源数据采集、热点事件识别、情感分析、趋势预测和智能问答。

## 系统架构

```
A 爬虫（Crawler）
  └─ 采集新闻/社交平台原始数据
        ↓
B 算法（Algorithm / NLP）
  └─ 清洗去重 → 事件聚类 → 情感分析 → 热度/生命周期预测
        ↓
C 后端（Backend API）
  └─ 整合 A/B 数据，提供 RESTful API
        ↓
D 前端（Frontend）
  └─ 事件看板 / 分析报告 / 智能问答
```

## 模块说明

| 目录 | 模块 | 职责 |
|------|------|------|
| `sina_crawler/` | A — 数据采集 | 新浪新闻爬虫，采集原始舆情数据 |
| `algo/` | B — 算法分析 | NLP 文本分析、事件聚类、情感分析、趋势预测 |
| `api-design/` | C — 接口契约 | 后端 API 路径与字段定义 |

## 主要功能

- **热点事件发现**：基于 TF-IDF 相似度的增量事件聚类
- **情感分析**：机器学习分类模型，区分评论与新闻正文
- **生命周期预测**：潜伏 / 成长 / 高潮 / 衰退四阶段判断
- **趋势图表**：报道量时间序列及未来走势预测
- **智能问答**：基于大模型的事件专项问答
